Use the **milestone** skill to manage GitHub milestones.

**Usage:**
- `/milestone` - 현재 마일스톤 확인 및 관리 메뉴
- `/milestone list` - 마일스톤 목록
- `/milestone create` - 새 마일스톤 생성
- `/milestone switch` - 현재 마일스톤 변경
- `/milestone close` - 마일스톤 종료
